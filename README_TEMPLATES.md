### 3. 🎨 TEMPLATE FIGMA (Wireframes e Design System)
**Arquivo:** `TEMPLATE_FIGMA.md`

#### Design System Incluso:
- **Paleta de Cores:** 10+ cores definidas
- **Tipografia:** Headlines, Body, Buttons
- **Componentes Base:**
  - Buttons (4 variações + 3 tamanhos)
  - Input Fields (6 tipos)
  - Cards
  - Badges
  - Modals

#### 7 Wireframes Principais:
1. **Login/Autenticação** - Tela de acesso
2. **Dashboard Principal** - Visão geral com KPIs
3. **Módulo Plantios** - Lista e CRUD de talhões
4. **Detalhes Plantio** - Visualização completa
5. **Módulo Rebanho** - Gestão de animais
6. **Dashboard Financeiro** - KPIs e gráficos
7. **Relatórios** - Download e visualização

#### Componentes Específicos para Agricultores:
- KPI Cards
- Alert Components
- Timeline de Operações
- Mapas de talhões

**Grid e Responsividade:**
- Desktop: 12 colunas
- Tablet: 8 colunas
- Mobile: 1 coluna

**Como usar:**
1. Abra o arquivo `TEMPLATE_FIGMA.md` neste repo
2. Abra seu Figma (figma.com)
3. Crie um novo projeto
4. Use o guia para criar componentes reutilizáveis
5. Monte as telas conforme os wireframes

## 💡 Dicas de Implementação

### PowerPoint
- Use as imagens recomendadas: fazendas modernas, drones, painéis solares
- Mantenha os padrões de cores em todos os slides
- Adicione animações subtis nas transições
- Tempo estimado: 40-50 minutos de apresentação

### Figma
- Crie componentes principais primeiro (Button, Input, Card)
- Depois monte as 7 telas principais
- Use Auto Layout para responsividade
- Exporte assets para o time de desenvolvimento

### PMBOK
- Use como guia para reuniões iniciais
- Atualize riscos e métricas conforme o projeto evolui
- Compartilhe com PM, Tech Leads e Stakeholders
